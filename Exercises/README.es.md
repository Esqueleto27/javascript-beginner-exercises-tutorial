# Desafíos de JavaScript para Principiantes

## Descripción

Este repositorio contiene una colección de **challenges de JavaScript para principiantes** diseñados para practicar los fundamentos más importantes del lenguaje.

El objetivo de estos ejercicios es fortalecer la comprensión de los conceptos clave de JavaScript mediante **práctica directa escribiendo código**. Cada ejercicio se enfoca en un tema específico como variables, tipos de datos, arrays, objetos o funciones.

En lugar de aprender solo teoría, vas a **resolver pequeños problemas de programación**, que es una de las mejores formas de aprender a programar.

---

## Cómo funcionan los ejercicios

Cada challenge dentro de este repositorio contiene algunos archivos principales:

### app.js

Este es el archivo principal de JavaScript donde escribirás tu código y resolverás el ejercicio.

### README.md

Este archivo contiene la explicación del desafío, el concepto que se está practicando y las instrucciones para completar el ejercicio.

### test.js

Algunos ejercicios pueden incluir pruebas automáticas que ayudan a verificar si tu solución funciona correctamente.

---

## Qué vas a practicar

Estos challenges están enfocados en los **fundamentos más importantes de JavaScript**, incluyendo:

- Variables
- Tipos de datos primitivos
- Constantes
- Tipado dinámico
- Objetos
- Arrays
- Funciones
- Argumentos de funciones
- Lógica básica de programación

Cada ejercicio aumenta gradualmente en dificultad para que puedas aprender paso a paso.

---

## Objetivo de este repositorio

El propósito de este repositorio es:

- Practicar los fundamentos de JavaScript
- Desarrollar pensamiento lógico
- Mejorar habilidades de resolución de problemas
- Construir una base sólida en programación

Estos ejercicios están pensados como **práctica tipo universidad**, donde primero entiendes un concepto y luego lo aplicas inmediatamente mediante programación.
