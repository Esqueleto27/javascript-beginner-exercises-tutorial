# JavaScript Challenge — Introducción

## Descripción

Este ejercicio es el inicio de una serie de **desafíos de JavaScript** diseñados para practicar los fundamentos del lenguaje.

JavaScript es uno de los lenguajes más utilizados en el desarrollo web. Se usa para agregar **lógica, interactividad y comportamiento dinámico** a las páginas web.

En este challenge comenzarás a familiarizarte con algunas de las herramientas básicas que todo programador de JavaScript utiliza.

---

## Conceptos que aprenderás

En este ejercicio comenzarás a trabajar con algunos conceptos fundamentales:

- `console.log`
- Concatenación de strings
- Arrays
- Funciones
- Loops
- Conceptos básicos de programación en JavaScript

Estos conceptos forman parte de la base que necesitas dominar para poder avanzar hacia temas más complejos como manipulación del DOM, APIs o frameworks como React.

---

## ¿Qué es `console.log`?

`console.log()` es una función de JavaScript que permite **mostrar información en la consola del navegador o del entorno de desarrollo**.

Los programadores la usan constantemente para:

- Verificar valores
- Depurar código
- Entender cómo se comporta un programa

Ejemplo:

```javascript
console.log("Hello World");
```
