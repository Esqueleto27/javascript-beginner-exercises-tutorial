# Challenge 01 — Hello World

## Introducción

Uno de los primeros pasos al aprender cualquier lenguaje de programación es imprimir un mensaje en pantalla. Tradicionalmente se utiliza el famoso ejemplo **"Hello World"**.

En JavaScript utilizamos la función `console.log()` para mostrar información en la consola.

La consola es una herramienta que permite a los programadores ver información mientras el programa se está ejecutando. Es muy útil para:

- Ver valores de variables
- Probar partes del código
- Depurar errores
- Entender cómo se comporta un programa

---

## ¿Qué es `console.log()`?

`console.log()` es una función que permite **imprimir información en la consola**.

Ejemplo:

```javascript
console.log("Hello World");
```
