# Challenge 01 — Hello World

## Introduction

One of the first steps when learning any programming language is printing a message on the screen. Traditionally, the classic example used is **"Hello World"**.

In JavaScript we use the function `console.log()` to display information in the console.

The console is a tool that allows developers to see information while the program is running. It is very useful for:

- Checking variable values
- Testing parts of the code
- Debugging errors
- Understanding how a program behaves

---

## What is `console.log()`?

`console.log()` is a function used to **print information in the console**.

Example:

```javascript
console.log("Hello World");
```
