# Challenge 02 — Print Variables to the Console

## Introduction

In JavaScript, **variables** are used to store information that can be used later in a program.

A variable can store different types of data such as:

- Strings
- Numbers
- Booleans
- Arrays
- Objects

Once a variable has a value, we can use `console.log()` to **display that value in the console**.

This is very useful to check if a variable contains the value we expect.

---

## How to Declare a Variable

In JavaScript there are several ways to declare variables, but the most common ones are:

- `let` → when the value can change
- `const` → when the value should not change

Example:

```javascript
let name = "John";
```
