# Desafío 02 — Imprimir Variables en la Consola

## Introducción

En JavaScript, las **variables** se utilizan para almacenar información que puede ser utilizada más adelante en el programa.

Una variable puede guardar diferentes tipos de datos como:

- Texto (strings)
- Números
- Booleanos
- Arrays
- Objetos

Una vez que una variable tiene un valor, podemos usar `console.log()` para **mostrar ese valor en la consola**.

Esto es muy útil para comprobar si una variable contiene el valor que esperamos.

---

## ¿Cómo declarar una variable?

En JavaScript existen varias formas de declarar variables, pero las más utilizadas son:

- `let` → cuando el valor puede cambiar
- `const` → cuando el valor no debe cambiar

Ejemplo:

```javascript
let name = "John";
```
