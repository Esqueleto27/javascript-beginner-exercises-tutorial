# JavaScript Beginner Challenges

## Overview

This repository contains a collection of **JavaScript beginner challenges** designed to practice the most important fundamentals of the language.

The goal of these exercises is to strengthen your understanding of core JavaScript concepts through **hands-on coding practice**. Each exercise focuses on a specific topic such as variables, data types, arrays, objects, or functions.

Instead of only reading theory, you will **solve small coding problems**, which is one of the best ways to truly learn programming.

---

## How the Challenges Work

Each challenge in this repository contains a small set of files:

### app.js

This is the main JavaScript file where you will write your code and solve the exercise.

### README.md

This file contains the explanation of the challenge, the concept being practiced, and the instructions for completing the exercise.

### test.js

Some exercises may include tests that help verify if your solution works correctly.

---

## What You Will Practice

These challenges focus on the most important **JavaScript fundamentals**, including:

- Variables
- Primitive data types
- Constants
- Dynamic typing
- Objects
- Arrays
- Functions
- Function arguments
- Basic programming logic

Each exercise is designed to gradually increase in difficulty so that you can build confidence step by step.

---

## Goal of This Repository

The purpose of this repository is to:

- Practice JavaScript fundamentals
- Develop logical thinking
- Improve problem-solving skills
- Build a strong programming foundation

These exercises are meant to simulate **university-style programming practice**, where you learn a concept and immediately apply it through coding.
