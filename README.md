# JavaScript Exercises

Este repositorio contiene una colección de ejercicios prácticos de JavaScript enfocados en reforzar los fundamentos del lenguaje.

Los ejercicios permiten practicar conceptos básicos como variables, condicionales, bucles y lógica de programación mediante pequeños retos de código.

El objetivo de este proyecto es mejorar la comprensión de JavaScript a través de la práctica constante y la resolución de problemas.
