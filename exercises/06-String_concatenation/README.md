# Concatenación de Strings en JavaScript

Este ejercicio practica la concatenación de strings utilizando el operador `+` en JavaScript.

El objetivo es asignar valores a las variables `myVar1` y `myVar2` para que, al unirlas, el programa imprima **"Hello World"** en la consola.

Este concepto es importante porque permite combinar textos y construir mensajes dinámicos dentro de un programa.
