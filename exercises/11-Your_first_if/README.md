# Uso de Condicionales (If) en JavaScript

Este ejercicio introduce el uso de **condicionales (`if`, `else if`, `else`)** para tomar decisiones dentro de un programa.

El programa recibe una cantidad de kilómetros restantes y, dependiendo del valor ingresado, muestra un mensaje diferente en la consola.

El objetivo es comprender cómo utilizar estructuras condicionales para controlar el flujo de ejecución de un programa según ciertos valores.
