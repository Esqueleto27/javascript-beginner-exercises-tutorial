# Imprimir Variables en la Consola

Este ejercicio introduce cómo declarar variables en JavaScript y cómo mostrar su contenido utilizando `console.log()`.

Se crea una variable llamada `color` a la que se le asigna el valor **"red"**, y posteriormente se imprime su valor en la consola.

El objetivo es comprender cómo almacenar información en variables y visualizar su contenido durante la ejecución del programa.
