# Uso de Constantes en JavaScript

Este ejercicio introduce el uso de **constantes** en JavaScript mediante la palabra reservada `const`.

El objetivo es comprender que una constante no puede cambiar su valor una vez declarada. En el ejercicio se debe identificar y corregir un error relacionado con la modificación de una constante para que el programa funcione correctamente.

Al corregir el problema, el programa debe mostrar **0.9** en la consola.
