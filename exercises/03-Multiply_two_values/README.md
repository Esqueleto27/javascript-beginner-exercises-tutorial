# Multiplicar Dos Valores en JavaScript

Este ejercicio practica el uso de operadores matemáticos en JavaScript, específicamente la multiplicación.

Se calcula el producto de **2345 × 7323**, se guarda el resultado en una variable llamada `variablesAreCool` y luego se muestra en la consola utilizando `console.log()`.

El objetivo es entender cómo realizar operaciones matemáticas y almacenar sus resultados en variables.
