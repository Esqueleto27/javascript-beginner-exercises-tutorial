# Crear una Nueva Función en JavaScript

Este ejercicio practica la creación y uso de funciones que reciben múltiples parámetros.

Se implementa la función `shortIntroduction()` que recibe **nombre, profesión y edad**, y devuelve una frase de presentación utilizando concatenación de strings.

El objetivo es comprender cómo pasar argumentos a una función y generar una salida dinámica con esa información.
