# Uso de Funciones en JavaScript

Este ejercicio introduce el uso de funciones en JavaScript para procesar datos y obtener resultados.

Se utiliza una función llamada `isOdd()` que permite verificar si un número es impar o par. El programa llama a la función pasando el número **45345** como argumento y muestra el resultado en la consola.

El objetivo es comprender cómo llamar funciones y trabajar con los valores que retornan.
